# Support

OpenSprout is a privacy-first plant care records app. Support is best-effort and focused on reproducible bugs, documentation, local development, and data-export concerns.

## Before Opening An Issue

- Check the [README](README.md), [SECURITY.md](SECURITY.md), and [CONTRIBUTING.md](CONTRIBUTING.md).
- Search existing issues and pull requests.
- Do not share personal records, private schedules, API keys, or database credentials.

## Good Issue Reports

- Include steps to reproduce and expected behavior.
- Describe whether the issue affects records, reminders, exports, or local setup.
- Include screenshots only after removing private information.

For security concerns, follow [SECURITY.md](SECURITY.md).
