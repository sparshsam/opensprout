# Security Policy

## Supported Versions

OpenSprout is pre-1.0 software. Security fixes are applied to the `main` branch until stable releases are introduced.

## Reporting a Vulnerability

Please report suspected vulnerabilities through GitHub Security Advisories for this repository.

If advisories are unavailable, open a minimal private report with:

- affected feature or file path
- steps to reproduce
- expected impact
- whether any user data may be involved

Please do not open public issues for active vulnerabilities until a fix is available.

## Response Expectations

- Acknowledgment target: 48 hours
- Initial triage target: 7 days
- Fix target for confirmed high-impact issues: 30 days

OpenSprout does not run a bug bounty program.
