# Roadmap

## v0.1 - Seedling

- Project scaffold and AGPLv3 license.
- Responsive PWA app shell.
- Plant dashboard, plant list, and plant detail panel.
- Add plant demo interaction.
- JSON export starter.
- Initial Supabase schema with RLS and Storage policies.

## v0.2 - Care Tracking

- Create/edit/delete plant profiles.
- Watering, fertilizing, pruning, repotting logs.
- Plant detail timeline.
- Filters by plant and care type.

## v0.3 - Reminders

- Reminder schedules per plant.
- Upcoming task dashboard.
- Mark complete, skip, and snooze.
- Timezone-aware date handling.

## v0.4 - Photos & Growth History

- Private photo uploads through Supabase Storage.
- Journal entries with photo metadata.
- Photo timeline and captions.
- Growth comparison view.

## v0.5 - Self-Hosting

- Dockerfile and Docker Compose.
- Backup and restore guide.
- Reverse proxy notes.
- Supabase local development guide.

## v0.6 - Data Ownership

- Full export/import flow.
- Backup manifest validation.
- Delete account/data flow.
- Privacy policy template for self-hosters.

## v0.8 - Community Garden

- Optional household/shared gardens.
- Community plant species database foundation.
- Contributor moderation workflows.

## v1.0 - Stable

- Stable schema and migration path.
- Accessibility and security pass.
- Complete self-hosting docs.
- Reliable PWA offline flow.
- Public demo and screenshots.
