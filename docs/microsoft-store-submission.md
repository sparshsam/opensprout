## Packages

- [ ] Generate MSIX via PWABuilder (`pwabuilder.com`) or manual `MakeAppx`

## Certification

- [ ] Complete age rating questionnaire (Everyone)
- [ ] Complete Data Safety section (see `docs/data-safety.md`)

- [ ] Submit for certification
