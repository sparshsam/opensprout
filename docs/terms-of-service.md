## 2. Description of Service

OpenSprout is a plant care tracking application that allows users to:

- Create and manage plant records
- Set up care schedules for watering, fertilizing, and other tasks
- Log completed care activities
- Maintain plant health journals with photos
- Export and import plant data
- Connect AI agents via MCP (Model Context Protocol) API tokens
- Identify plants through the PlantNet API integration
- Receive care reminders (optional, opt-in)
