# OpenSprout — Roadmap

> Privacy-first plant care software for schedules, logs, and stewardship.

## Current — v0.1 (June 2026)

Initial foundation.

- [x] Plant profiles and care schedules
- [x] Care logging and tracking
- [x] Reminder notifications
- [x] Privacy-first design (no unnecessary data collection)
